Purpose : This Doc Specifies the basic requirements for the IT Resource Project. Please add the 
	       Requirements for the project. Seperatly add Basic and Nice To Have Requirements 

Basic Requirements :

1. Custom Login / Signup Page
2. Signup with Google Plus , Facebook etc
3. Merge custom accounts and accounts made through google plus , facebook etc
4. Users are authorized to access application features based on their roles.
5. Admin will verify the managers and managers will verify the employee that explains the hierarchy.
6. User can request for IT Resource .
7. There will be priority of each request assigned by the manager.
8. Helpdesk will assign the resource and the status for the request.

Nice To Have :
	
1. Decide efficiency based on the time required to complete the request.
2. After completion of the request , the employee can add the satisfaction level.
3. Option to raise a missing ticket for a resource not allotted.
